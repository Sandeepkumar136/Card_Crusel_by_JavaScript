# Review carusel 

In this tutorial, i  learned how to create a carousel of reviews with a button that generates random reviews.

This is a good feature to have on an ecommerce site to display customer reviews or a personal portfolio to display client reviews.
## Key concept covered

- objects
- DOMContentLoaded
- addEventListener
- array.length
